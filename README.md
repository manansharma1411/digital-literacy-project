## Project Overview

This repository contains my **Digital Literacy Portfolio Project**, created as part of the CSE0001 course at VIT Bhopal University. The project is based on the role of a *Student Digital Ambassador*, where the objective is to help understand and effectively use digital tools, maintain a professional online presence, and stay safe in the digital environment. 

The portfolio is divided into five modules, each focusing on a key aspect of digital literacy, including awareness, professional presence, collaboration tools, communication, and cybersecurity.


## Module Summaries

### Module 1: Digital Literacy Awareness

Created a visual infographic explaining the concept of digital literacy, its importance, and safe internet practices using tools like Canva.

### Module 2: Student Digital Portfolio

* GitHub
